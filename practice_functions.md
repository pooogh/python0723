## Блок 1. Функции, Лямбды и Тернарный оператор

### Теория

**1. Базовые функции и возврат нескольких значений**
Функции в Python могут принимать аргументы по умолчанию и возвращать сразу несколько значений. Под капотом Python упаковывает такие возвращаемые значения в кортеж (tuple).

`python
def pow(x, y=2):
    return x ** y

def find_mx_mn(numbers):
    # Возвращает два значения в виде кортежа (max_val, min_val)
    return max(numbers), min(numbers) 
`

**2. Анонимные функции (lambda)**
Иногда функция нужна ровно на один раз (например, для сортировки или фильтрации). Вместо `def` можно использовать короткие `lambda` функции. Они пишутся в одну строку и не требуют слова `return`.

`python
# Обычная функция
def norm(string):
    return string.lower().replace('  ', ' ')

# То же самое через lambda
norm_2 = lambda s: s.lower().replace('  ', ' ')

# Можно сразу вызвать (самовызывающаяся функция)
x = 3
print((lambda x: x ** 2)(x)) # Выведет 9
`

**3. Тернарный оператор**
Это способ записать условие `if-else` в одну строку. Очень удобно использовать внутри лямбда-выражений или при присваивании.

`python
y = 2
# Синтаксис: [значение_если_True] if [условие] else [значение_если_False]
x = 1 if y > 2 else 3 
`

## Блок 2. Упаковка и распаковка данных (*args, **kwargs)

### Теория

Символы `*` и `**` в Python — это мощные инструменты для работы с коллекциями. Они позволяют "распаковывать" элементы из списков/словарей или, наоборот, "упаковывать" лишние аргументы функции в одну переменную.

**1. Базовая распаковка переменных**
`python
x, y = [1, 2]
dict_6 = {'k1': 'v1', 'k2':'v2'}
key1, key2 = dict_6 # В переменные попадут КЛЮЧИ словаря

# Можно игнорировать ненужные значения с помощью _
x, _, y = [1, 2, 3] # x=1, y=3
`

**2. Упаковка в аргументах функции**
Если мы не знаем, сколько аргументов передадут в функцию, мы используем `*` (собирает позиционные аргументы в кортеж) и `**` (собирает именованные аргументы в словарь).

`python
def pack(x, *tail):
    pass
pack(1, 2, 3, 4, 5) # x = 1, tail = (2, 3, 4, 5)

# Важный нюанс: если аргумент идет ПОСЛЕ звездочки, 
# при вызове его нужно передавать строго по имени (keyword-only):
def pack_2(*head, x):
    pass
pack_2(1, 2, 3, 4, x=5) # head = (1, 2, 3, 4), x = 5

def pack_dict(**dictionary):
    return dictionary
pack_dict(key="value", key2="value2") # Вернет словарь {'key': 'value', 'key2': 'value2'}
`

**3. Слияние коллекций**
С помощью `*` (для списков) и `**` (для словарей) можно легко объединять данные:
`python
nums_1 = [1, 2, 3]
nums_2 = [4, 5, 6]
nums_3 = [*nums_1, *nums_2] # [1, 2, 3, 4, 5, 6]

dict_7 = {'k1': 'v1'}
dict_8 = {'k2': 'v2'}
dict_9 = {**dict_7, **dict_8} # {'k1': 'v1', 'k2': 'v2'}
`

## Блок 3. Функции высшего порядка: map, filter, reduce

### Теория

Функции высшего порядка (ФВП) принимают другие функции в качестве аргументов. Они не изменяют исходный массив, а создают новые специальные объекты-итераторы (поэтому для вывода их часто оборачивают в `list()`).

**1. Импорты модулей**
Перед тем как использовать сложные функции (например, `reduce`), нужно подключить соответствующий модуль. Это можно сделать разными способами:
`python
import functools # Вызов: functools.reduce()
import functools as f # Вызов: f.reduce()
from functools import reduce # Вызов напрямую: reduce()
`

**2. map(func, iterable)**
Применяет функцию `func` к каждому элементу последовательности `iterable`.
`python
str_to_int = ['1', '2', '3']
# Преобразуем строки в числа:
map_3 = list(map(int, str_to_int)) # [1, 2, 3]
`

**3. filter(func, iterable)**
Отбирает только те элементы, для которых `func` вернула `True`.
`python
to_filter = [1, 2, 3, 4, 5, 6, 7]
only_odd = list(filter(lambda x: x % 2 != 0, to_filter)) # [1, 3, 5, 7]
`

**4. reduce(func, iterable, [initial])**
Сворачивает массив в одно значение. Функция `func` всегда принимает два аргумента: аккумулятор (накопленный результат) и текущий элемент.
`python
from functools import reduce

to_reduce = [1, 2, 3]
# 1 шаг: acc=0 (по умолчанию первый элемент), item=2 -> 1+2=3
# 2 шаг: acc=3, item=3 -> 3+3=6
sum_of_arr = reduce(lambda acc, item: acc + item, to_reduce) 

# Пример сложнее: вычисление факториала (n!)
def fact_2(n):
    return 1 if n == 0 else reduce(lambda x, y: x * y, range(1, n + 1), 1)
`
